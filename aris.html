Magang di PT. Universal Big Data Malang memberi saya kesempatan untuk mengembangkan berbagai proyek

web teknologi. Saya bekerja pada:

1. SiakadPlus: Sistem manajemen sekolah.

2. Botpress Cloud: Membuat beberapa chatbot:

  • Bot Movie: Rekomendasi film.
  • Bot Pelanggaran Siswa: Mencatat dan mengelola pelanggaran siswa.
  • Bot BPJS: Informasi layanan BPJS.
  • Bot Quiz Login dengan OTP: Kuis dengan login OTP.
  • Bot Laporan Masyarakat: Pelaporan oleh masyarakat.

    • Selain itu, saya menggunakan Django dan React.js untuk input pelanggaran siswa serta Frappe Framework untuk aplikasi tes magang,

    Ubig LSM, dan Education. Laporan ini akan menjelaskan setiap proyek dan teknologi yang digunakan.

    Teknologi Yang Digunakan

    Laravel: Framework PHP yang digunakan untuk mengembangkan backend SiakadPlus.

    Flutter: Framework UI untuk membangun antarmuka pengguna SiakadPlus yang responsif di berbagai platform.

    Django: Framework web yang digunakan untuk mengembangkan aplikasi pengiriman pesan WhatsApp melalui API Nobox.

    React.js: Library JavaScript untuk membangun antarmuka pengguna yang dinamis dalam proyek Django.

    Nobox API: Sistem yang digunakan untuk integrasi dan mengirim pesan melalui WhatsApp.

    Django REST Framework: Toolkit untuk membangun API web menggunakan Django.

    MySQL: Sistem manajemen basis data relasional yang digunakan untuk menyimpan data aplikasi.

    Python: Bahasa pemrograman utama yang digunakan dalam pengembangan backend.

    Frappe Framework: Framework yang digunakan untuk membuat aplikasi seperti tes magang, Ubig LSM, dan Education

    Ubuntu: Sistem operasi yang mendukung pengembangan menggunakan Frappe Framework

    Docker: Platform untuk mengelola kontainer aplikasi (opsional)

    Django Pelanggaran Siswa

    Proyek Django ini mencatat pelanggaran siswa dan mengirim notifikasi

    otomatis melalui email dan WhatsApp menggunakan API Nobox.

    Aplikasi ini mendukung banyak sekolah dengan penginputan

    pelanggaran oleh guru, memastikan komunikasi yang cepat dan efektif

    dengan wali siswa.

    React Js Pelanggaran Siswa

    Frontend proyek ini dibangun dengan React JS dan terhubung dengan

    API Django untuk manajemen pelanggaran siswa. Aplikasi ini

    menyediakan menu siswa, kategori, dan pelanggaran, memungkinkan

    pengguna untuk mengelola data pelanggaran siswa secara efisien di

    sekolah yang menggunakan aplikasi ini.
